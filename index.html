Vamos a profundizar nuestra habilidad en la etiqueta "a" de HTML aprendiendo a crear enlaces usando URL absolutas y relativas. Para esta actividad, podrás guiarte del cuadernillo cuyo enlace está arriba.

1.1- ¿qué es una URL?

Uniform Resource Locator (URL) es una cadena de texto que especifica dónde puede encontrarse un recurso (página, imagen, video, sonido, etc.) en Internet. Es también conocido como dirección WEB o enlace.

1.2- ¿qué es una url absoluta?

Una URL absoluta es el camino de directorios a seguir separado por barras al documento solicitado.

1.3- ¿qué es una url relativa?

Son estructuras de URL que contienen la ruta de ubicación, omitiendo el nombre del sitio web.

1.4- ¿cuál es la diferencia entre una url relativa y una url absoluta?

Una esta completa y la otra no.

1.6- ¿qué es un directorio? ¿qué relación hay con una carpeta?

Un directorio es un tipo exclusivo de archivo que sólo contiene la información necesaria para acceder a archivos o a otros directorios. Carpeta es otra manera de refrirse a los directorios.

1.7- En la estructura de un proyecto web, ¿a qué se llama directorio raíz? ¿Con qué símbolo se representa?

El directorio raíz es el primer directorio o carpeta en una jerarquía. Se representa mediante el símbolo de barra inclinada (/).

1.10- ¿Cuál es el propósito de la etiqueta anchor?

La etiqueta a se llama Anchor. Nos permite crear un enlace (hipervínculo) a otra página web y/o un recurso ubicado en el mismo proyecto u en otro servidor web.

1.11- ¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

Se utiliza el href.

1.12- ¿Qué sucede si el atributo "href" en la etiqueta anchor está vacío?

No te enlaza con ningun vinculo.

1.13- ¿Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

Especifica en donde mostrar la URL enlazada. Esto dependerá del contexto del navegador, siendo posible: pestaña, ventana, iframe.

1.14- Entonces ¿qué es un hipervínculo?

Un hipervínculo es un enlace, es una referencia unidireccional en un documento electrónico que entrelaza diferentes documentos o secciones entre sí.
